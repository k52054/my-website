# Cat Cattington's Personal Website

A simple, fun profile page dedicated to Cat Cattington and all things feline. This project is styled using Tailwind CSS to showcase clean layout design and responsive web elements.

### Features
- Fully styled with Tailwind CSS utility classes
- Centered layout for readability across different screen sizes
- Features high-quality placeholders of our favorite furry friends
